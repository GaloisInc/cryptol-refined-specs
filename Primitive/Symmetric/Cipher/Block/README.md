This directory includes executable specifications for Block Ciphers.

## AES

The refined cryptol specification presented here are currently being used in the [rustcrypto-verification](https://github.com/GaloisInc/rustcrypto-verification/tree/main) repository where you will verifications of AES's encryption and decryption algorithms for differeht key sizes (128, 192 and 256) of the rust [aes](https://crates.io/crates/aes/0.8.4) crate against the refined specs.
